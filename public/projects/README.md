# Imagens dos Projetos

Para adicionar imagens dos seus projetos, coloque os arquivos nesta pasta com os seguintes nomes:

1. `devburger.jpg` - Imagem do projeto DevBurger
2. `sistema-gestao.jpg` - Imagem do Sistema de Gestão Empresarial  
3. `hsk-tecnologia.jpg` - Imagem da HSK Tecnologia
4. `ecommerce-shoes.jpg` - Imagem do E-commerce de Calçados

Recomendações:
- Formato: JPG ou PNG
- Tamanho: 800x500px (proporção 16:10)
- Qualidade: Boa resolução para web
- Peso: Máximo 500KB por imagem

Depois de adicionar as imagens, elas serão automaticamente usadas no portfólio.
